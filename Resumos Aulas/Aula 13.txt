Repetições
- Estruturas de Repetições são um bloco de comando e códigos que são executados repetidademente até que uma condição se torne falsa, ou o numero estipulado de repetições se atinja;

while(condição){

    // Todos os comandos aqui serão repetidos até a condição se tornar falsa

}

Nesse método acima, ele primeiro testa, depois executa;
Existe também um método onde primeiro executa, depois testa, o 'do while';

do{

  // Aqui ele vai executar os comandos

} while(condição);
