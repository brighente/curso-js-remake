Condições
- São 'desvios' do fluxo padrão do código;

	if(condição){

		true
	
	} else {

		false

 	}

- Usando apenas o if, sem o else, temos uma Condição Simples, onde não acontece nada caso a condição não seja verdadeira;

	if(condição){

		true

	}
